Setup Instructions

You're seeing an "Unauthorized access" error because the Google Apps Script needs to be properly deployed. Follow these steps:

  1. Go to https://script.google.com and create a new project
  2. Copy the code from docs/google-apps-script-example.js into the script editor
  3. Click Deploy > New deployment
  4. Select Web app as the deployment type
  5. Set Execute as to "Me" (your Google account)
  6. Set Who has access to "Anyone" (this is essential!)
  7. Click "Deploy" and authorize when prompted
  8. Copy the provided web app URL and paste it below
